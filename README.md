# zip-utility
Zip utility class

-by Sujeet <sujeetkv90@gmail.com>

Install using composer:

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sujeetkv/zip-utility"
        }
    ],
    "require": {
        "sujeetkv/zip-utility": "dev-master"
    }
}
```
